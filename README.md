# Advanced Portfolio Website

## Project Overview

This project is an advanced responsive portfolio website built using HTML5, CSS3, CSS Grid, Flexbox, animations, and JavaScript.

## Features

- Responsive Design
- CSS Grid Layout
- Flexbox Components
- Dark Mode Toggle
- Smooth Animations
- Mobile First Design
- Modern UI Design

## Technologies Used

- HTML5
- CSS3
- JavaScript
